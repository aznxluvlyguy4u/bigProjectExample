# NSFO API
## Installation
Install Redis

```
$ sudo apt-get install redis
```

Install Postgres

```
$ sudo apt-get install postgres-9.3
```

Install PHP postgres pdo driver

```
$ sudo apt-get install php5-pgsql
```

## Configuration
Install composer dependencies

```
$ composer install
```

## Run

```
$ php bin/console server:start
```
